# RPress

RUST replacement version of Wordpress.

## Technology Stack

- Rocket （Web Framework）
- Askama or Tera or handlebars-rust (Template engine)
- Diesel (DataBase ORM)
- PostgreSQL (DataBase)
- jsonwebtoken + bcrypt (Auth)
- Redis (KVS)
- Yew or Seed (UI)
- Docker (container)
